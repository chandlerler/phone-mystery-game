# 雾港来信更新日志

本项目采用两位小数版本号。每次发布新的功能更新时，版本号增加 `0.01`，并在此文件顶部记录日期与改动内容。

## v1.06 - 2026-07-18

- 新增消息提示音和震动反馈，角色来信与玩家回复使用不同的反馈节奏。
- 新增“设置”应用，可分别开启或关闭消息提示音、消息震动，并保存到游戏存档。
- 下拉通知栏增加提示音和震动快捷开关，与“设置”应用保持实时同步。
- 微讯消息数据新增图片和内部链接类型，同时继续兼容原有纯文字剧情与旧存档消息。
- 林夏的首段 Demo 对话加入码头照片、朋友圈动态和浏览器文章，可从聊天中直接跳转。
- 浏览器搜索结果支持打开文章阅读页；版本更新为 v1.06。

## v1.05 - 2026-07-18

- 重写微讯消息投递：剧情节点触发后先进入待查看状态，玩家打开对应人物聊天才会开始逐条发送。
- 消息输入时间根据文本长度动态计算，长消息等待更久；人物头像旁只显示“正在输入中”，不再显示同步时间和信任度。
- 玩家选择回复后，角色的后续回应也会逐条发送，全部发送完毕后才触发下一剧情节点。
- 修复移动端下拉通知栏，增加 Pointer Events 与 Touch Events 双重手势支持，并扩大顶部可下拉区域。
- 主菜单新增醒目的“继续游戏”按钮，已有会话时同时显示右上角返回按钮。
- 触屏手机改为真正全屏模式，隐藏模拟外框、圆角、阴影和刘海；电脑端继续保留手机模型外观。
- 新增可配置的分段剧情时钟：阶段内按现实时间等速推进，到阶段上限停止，关键事件后跳到下一阶段。
- Demo 时间轴配置集中保存在 `data/characters.js`，正式剧本可以直接替换阶段时间与触发条件。
- 版本更新为 v1.05，并启用独立的新存档结构。

## v1.04 - 2026-07-17

- 将故事升级为《回声盲区》：加入林夏、周衡、唐梨、许舟、顾闻川五名可互动人物，并为每人配置本地头像。
- 新增 18 组长篇对话、30 个交叉剧情节点、信任值、未读消息、选择记忆与人物立场变化。
- 新增相册、录音、档案、朋友圈、浏览器、密钥库等多类线索，谜题结果会触发新消息与后续调查。
- 新增六种结局，由证据数量、人物关系、发布方式和最终选择共同判定。
- 按 `data/characters.js`、`dialogues.js`、`clues.js`、`events.js`、`endings.js` 拆分剧情数据，并将运行逻辑迁移到 `game.js`。
- 建立 `assets/avatars`、`assets/photos`、`assets/videos` 资源目录，加入五名人物头像、关键照片及视频封面。
- 更新一键发布脚本，使其自动同步 HTML、游戏引擎、剧情数据和全部媒体资源。
- v1.04 使用新的存档结构；旧版本进度不会覆盖新剧情存档。

## v1.03 - 2026-07-17

- 为林夏加入真人头像，并应用到锁屏通知、下拉通知栏、桌面通知、聊天联系人、聊天气泡和朋友圈。
- 头像来自 Unsplash 免费授权图片，已下载到项目本地，避免外链失效或图库响应影响游戏。
- 新增 `outputs/assets` 图片资源目录和来源说明文档，方便后续继续加入剧情图片、照片和视频封面。
- 发布脚本现在会自动同步整个 `assets` 目录到 GitHub，后续本地增加图片后仍可一键发布。
- 将头像优化为适合手机加载的 640 像素版本，文件约 82 KB。

## v1.02 - 2026-07-17

- 为微讯、相册、文档、浏览器、朋友圈、视频、私密箱、结局和帮助设计对应的应用 Logo。
- 图标采用项目内置 Lucide 矢量路径并直接嵌入 HTML，不依赖外部 CDN。
- 统一桌面、Dock、锁屏通知、下拉通知栏、消息动画和应用标题栏的图标语言。
- 为九个应用保留独立配色，提高手机桌面上的辨识度。
- 优化视频、私密箱、结局和帮助的图标底色，减少界面单一色调。

## v1.01 - 2026-07-17

- 新增游戏初始页面，打开链接后首先进入《雾港来信》主菜单。
- 新增“开始游戏”“保存游戏”和“读取游戏”功能。
- 新增存档时间和存档版本信息，兼容 v1.00 已有的浏览器本地进度。
- 新增状态栏游戏菜单入口，进入游戏后可以随时返回主菜单保存或读取。
- 主菜单显示制作者 `chandlerler` 与当前版本号 `v1.01`。
- 增加手机全面屏、安全区域和软键盘适配，修复部分安卓浏览器内容层高度塌缩的问题。
- 增加基础页面缓存控制，降低手机浏览器继续显示旧版本的概率。
- 新增一键 GitHub 发布脚本，自动同步游戏页面与更新日志。

- 实现模拟手机锁屏、密码解锁、桌面、通知栏和应用切换。
- 实现微讯、相册、文档、浏览器、朋友圈、视频、私密箱、结局和帮助应用。
