# 雾港来信更新日志

本项目采用两位小数版本号。每次发布新的功能更新时，版本号增加 `0.01`，并在此文件顶部记录日期与改动内容。

## v1.02 - 2026-07-17

- 为微讯、相册、文档、浏览器、朋友圈、视频、私密箱、结局和帮助设计对应的应用 Logo。
- 图标采用项目内置 Lucide 矢量路径并直接嵌入 HTML，不依赖外部 CDN。
- 统一桌面、Dock、锁屏通知、下拉通知栏、消息动画和应用标题栏的图标语言。
- 为九个应用保留独立配色，提高手机桌面上的辨识度。
- 优化视频、私密箱、结局和帮助的图标底色，减少界面单一色调。

## v1.01 - 2026-07-17

- 新增游戏初始页面，打开链接后首先进入《雾港来信》主菜单。
- 新增“开始游戏”“保存游戏”和“读取游戏”功能。
- 新增存档时间和存档版本信息，兼容 v1.00 已有的浏览器本地进度。
- 新增状态栏游戏菜单入口，进入游戏后可以随时返回主菜单保存或读取。
- 主菜单显示制作者 `chandlerler` 与当前版本号 `v1.01`。
- 增加手机全面屏、安全区域和软键盘适配，修复部分安卓浏览器内容层高度塌缩的问题。
- 增加基础页面缓存控制，降低手机浏览器继续显示旧版本的概率。
- 新增一键 GitHub 发布脚本，自动同步游戏页面与更新日志。

## v1.00 - 2026-07-17

- 完成《雾港来信》最小可玩原型。
- 实现模拟手机锁屏、密码解锁、桌面、通知栏和应用切换。
- 实现微讯、相册、文档、浏览器、朋友圈、视频、私密箱、结局和帮助应用。
- 完成“聊天 → 相册 → 浏览器 → 文档 → 私密箱”的基础谜题链。
- 使用浏览器本地存储自动保存解谜进度。
